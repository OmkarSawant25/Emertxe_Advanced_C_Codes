Decide rows and columns at runtime:

Allocate array of row pointers:

Allocate memory for each row individually:

for(int i = 0; i < rows; i++)

Access or modify elements using:

Free memory after use:

for(int i = 0; i < rows; i++)
    free(arr[i]);
free(arr);
